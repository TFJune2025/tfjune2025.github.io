# tfjune2025.github.io

I build practical AI systems meant to be used, with a focus on computer vision, autonomous MLOps, and operationally disciplined tooling. In parallel, I am building a hands-on path into electrical engineering through embedded systems, physical interfaces, and enclosure design.

This repository is the umbrella portfolio site for my work across:
- applied AI and computer vision
- autonomous MLOps and workflow automation
- agentic CLI / workflow automation
- embedded / hardware learning projects
- public-facing demos and technical writeups

Current featured hardware builds include:
- **LED Pomodoro Timer** — an Arduino Mega–based timer with a custom case designed in LightBurn
- **Казкар** — a simple three-button MP3 player built on Arduino Uno, designed for children to listen to audiobooks without the distractions of screens or extra interface complexity

## Structure
- `index.html` — landing page
- `styles.css` — site styling
- `script.js` — dynamic project-card rendering
- `data/projects.json` — featured-project data
- `assets/` — images and static assets
